SAÉ 1.04 - Développement Web & Identité Numérique (Projet IronPulse)

Conception et développement d'un site web vitrine responsive.

Contexte & Objectifs

Projet : Conception et développement de 'IronPulse', un site web multipages vitrine pour une salle de sport fictive.

Objectif : Créer une présence numérique moderne, intuitive et responsive pour promouvoir des services et équipements sportifs.

Mon Rôle : Développeur Fullstack (Front-end). J'ai géré l'intégralité du code (HTML/CSS), le design et le déploiement.

Compétences & Apprentissages Critiques (AC)

AC13.04 : Architecture Web

Connaître l’architecture et les technologies d’un site Web (Validé par la structure HTML5 sémantique et le style CSS3).

AC13.01 : Outils Informatiques

Utiliser un système informatique et ses outils (Validé par l'utilisation de Git et GitHub Pages pour le versioning et l'hébergement).

Réalisation & Tâches Techniques

  • Architecture & Code :
    • Développement de la structure en HTML5 (Pages Accueil, Services, Coachs, Contact).
    • Mise en page avancée en CSS3 utilisant Flexbox et CSS Grid pour un design moderne.
  • Qualité & Accessibilité :
    • Site entièrement Responsive (Media Queries) pour s'adapter aux mobiles et tablettes.
    • Respect des normes d'accessibilité WCAG 2.0 AA (contrastes, balises sémantiques).
    • Validation du code via le W3C Validator.
  • Déploiement (DevOps) : Hébergement public via GitHub Pages.

Preuves (Traces)

Page d'accueil du site IronPulse (Responsive Design)

Extrait de code CSS montrant l'utilisation de Flexbox et des Media Queries

Autoévaluation

Apprentissage : J'ai appris à ne pas seulement "faire marcher" un site, mais à le rendre propre et accessible. La séparation stricte entre le fond (HTML) et la forme (CSS) est acquise.

Recul : Bien que mon cœur de métier soit le réseau, comprendre le fonctionnement d'un service web (HTTP, DNS, Hébergement) est crucial pour savoir l'administrer plus tard.
